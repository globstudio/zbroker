zpipes_agent(3)
==============

NAME
----
zpipes_agent - work with background zpipes agent

SYNOPSIS
--------
----
//  Create a new zpipes_agent
zpipes_agent_t *
    zpipes_agent_new (void);

//  Destroy the zpipes_agent
void
    zpipes_agent_destroy (zpipes_agent_t **self_p);

//  Self test of this class
int
    zpipes_agent_test (bool verbose);
----

DESCRIPTION
-----------

The zpipes_agent class starts a background zpipes agent.


EXAMPLE
-------
.From zpipes_agent_test method
----
    //  Simple create/destroy test
    zpipes_agent_t *self = zpipes_agent_new ();
    assert (self);
    zpipes_agent_destroy (&self);
----
